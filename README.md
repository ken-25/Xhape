## 🚀 概要

**Xhape** は、WPFやWinUIで使用される **XAML Geometry (StreamGeometry)** をブラウザ上で視覚的に作成・編集するためのツールです。直感的なマウス操作で図形を描画し、リアルタイムに生成される XAML コードをコピー＆ペーストしてプロジェクトに統合できます。
https://ken-25.github.io/Xhape/

## ✨ 主な機能

- 🎨 **ビジュアルドローイング**: Line, Rectangle, Circle, Path をサポート。
- 🔄 **双方向同期**: キャンバス上の編集と XAML コードの直接編集がリアルタイムに同期。
- 🎯 **精密な編集**: グリッドスナップ、Shiftキーによる 90 度補正。
- 🔍 **自在なビューポート**: マウスホイールによるズーム、ドラッグによるパン操作。
- 🛠️ **セレクトモード**: 配置済みオブジェクトの移動・リサイズ・削除。
- 🌓 **ダークモード**: 長時間の作業でも目に優しいモダンな UI。

## 🛠️ クイックスタート

1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開きます。
3. すぐに描画を開始できます！

## ⌨️ ショートカットキー

| キー | 機能 |
| :--- | :--- |
| `S` | Select ツール |
| `L` | Line ツール |
| `R` | Rectangle ツール |
| `C` | Circle ツール |
| `P` | Path ツール |
| `G` | グリッドスナップの ON/OFF |
| `Shift` + 描画 | 水平・垂直に固定（90度スナップ） |
| `Delete` / `Backspace` | 選択中の図形を削除 |
