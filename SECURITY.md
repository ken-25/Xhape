# Security Policy

## Supported Versions

Currently, only the latest version of Xhape is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT create a public issue. Instead, report it by following these steps:

1. Send an email to [INSERT EMAIL ADDRESS].
2. Provide a detailed description of the vulnerability and steps to reproduce it.
3. We will acknowledge your report and work on a fix.

Thank you for helping keep Xhape secure!
